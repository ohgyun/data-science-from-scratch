# 밑바닥부터 시작하는 데이터 과학

[밑바닥부터 시작하는 데이터 과학(Data Science from Scratch)](http://book.naver.com/bookdb/book_detail.nhn?bid=10652749)을 읽으며 정리한 학습 노트입니다.

## Requirements
- Docker

## Install & Run
Jupyter 노트북을 Docker 컨테이너로 실행할 수 있게 설정해뒀습니다. Docker 설치 후, 아래 스크립트만 실행하시면 됩니다.
```
$ ./run_notebook.sh
```
