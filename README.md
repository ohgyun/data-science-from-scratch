# 데이터 과학 학습 노트

데이터 과학을 공부하며 정리해둔 주피터 노트북입니다.

## Requirements
- Docker

## Install & Run
Jupyter 노트북을 Docker 컨테이너로 실행할 수 있게 설정해뒀습니다. Docker 설치 후, 아래 스크립트만 실행하시면 됩니다.
```
$ ./run_notebook.sh
```
